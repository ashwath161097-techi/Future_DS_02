# 📊 Customer Churn & Retention Analytics

An interactive Power BI dashboard project focused on analyzing customer churn, retention behavior, and business performance using Excel and Power BI.

---

## 🚀 Project Overview

This project helps businesses understand:

- Customer churn patterns
- Customer retention metrics
- Revenue impact analysis
- Customer segmentation
- Business KPIs and trends

The dashboard provides actionable insights to improve customer retention strategies and business decision-making.
